# Mini Market 🛒

A Laravel 11 mini-market application where you can:
- Add new products
- See all products
- Search and filter
- Sort by name, price, stock, or date
- View product details
- Edit or delete products

## 🚀 Tech Stack
- PHP 8.3
- Laravel 11
- MySQL (via Laragon)
- Bootstrap 5 (UI)
- Blade templates
